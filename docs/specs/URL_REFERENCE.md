## 注意事項

1. **動的ルート**: `[id]` を含むURLは、実際のIDに置き換えてアクセスしてください
   - 例: `/ja/consumer/history/tx_001`
   - 例: `/ja/enterprise/users/user_001`

2. **言語切り替え**: URLの `ja` を `en` に変更することで英語版にアクセスできます

3. **開発サーバー起動**:
   ```bash
   cd apps/web
   pnpm dev
   ```

4. **ポート変更時**: 環境変数 `PORT` でポートを変更した場合は、URLのポート番号も変更してください
