## 1. Overview

### 1.1 プロジェクト概要

Quantum Shieldは、量子コンピュータ時代に向けた資産保護プロトコル。従来のECDSA秘密鍵に依存せず、Dilithium署名とSPHINCS+署名による二重認証で資産を守る。

### 1.2 Core Principles

| ID | 原則 | 説明 |
|----|------|------|
| CP-1 | 自己主権性 | 常にユーザーが自身の資産をコントロールできる |
| CP-2 | 量子耐性 | NIST標準PQCで将来の量子攻撃に備える |
| CP-3 | 透明性 | すべての操作がオンチェーンで検証可能 |
| CP-4 | 分散化 | 単一障害点を排除し、検閲耐性を持つ |
| CP-5 | 持続可能性 | 長期運用可能な経済モデル |

### 1.3 アプリ一覧
