Frequently Asked Questions

How do I rotate my API keys? +
Navigate to API Keys section, click "Rotate" on the key you want to rotate. A new key will be generated and the old one will remain active for 24 hours.
What happens during an emergency unlock? +
Emergency unlocks bypass the normal 24-hour timelock but require a 7-day waiting period and incur a 5% penalty fee.
How do I add IP addresses to the allowlist? +
Go to Settings > Security, enable IP Allowlist, and add your IP addresses. Changes take effect immediately.
