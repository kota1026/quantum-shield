Emergency Pause

⏸️

System Pause

全てのLock/Unlock操作を即座に停止します

⚠️ 重要な注意事項
Pauseは最大72時間有効です。
実行前に必ずSecurity Councilメンバーと協議してください。
全ての操作は監査ログに記録されます。

Pre-Pause Checklist

インシデントの深刻度を確認した
Security Councilメンバーに連絡した
影響範囲を把握した（TVL、ユーザー数）
コミュニティ通知の準備ができている
復旧計画の概要がある

Recovery Procedures

1
根本原因の特定
インシデントの原因を調査し、文書化する
2
修正の実装
必要なパッチまたは設定変更を適用
3
テスト環境での検証
修正が正しく機能することを確認
4
Security Council承認
復旧の承認を得る（マルチシグ）
5
System Resume
Pauseを解除し、監視を継続
6
ポストモーテム作成
インシデントレポートを公開
