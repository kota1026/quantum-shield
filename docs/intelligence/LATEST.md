## Setup Required

To enable fully autonomous weekly research:

1. Get an Anthropic API key from https://console.anthropic.com
2. Add it as a GitHub Secret: `ANTHROPIC_API_KEY`
3. The weekly research workflow runs every Monday at 9:00 UTC
4. Results appear as PRs in the repository

## Manual Research

In any Claude Code session, research runs automatically at session start (see CLAUDE.md "Autonomous Research & Strategy" section).
