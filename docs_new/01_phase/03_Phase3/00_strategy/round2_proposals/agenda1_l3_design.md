```
┌─────────────────────────────────────────────────────────────┐
│                   DA Strategy                                │
├─────────────────────────────────────────────────────────────┤

│  ├── State Roots                                           │

│  │              DA Selection Matrix                     │   │
│  │                                                      │   │
│  │  Data Type      │ L1 (Ethereum) │ Alt-DA (Celestia) │   │
│  │  ─────────────────────────────────────────────────   │   │
│  │  State Root     │      ✅       │        ❌         │   │
│  │  STARK Proof    │      ✅       │        ❌         │   │
│  │  TX Data        │      ❌       │        ✅         │   │
│  │  Historical     │      ❌       │        ✅         │   │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 要件 | 仕様 | 根拠 |
|------|------|------|
| Critical Data | Ethereum L1 | セキュリティ最優先 |

| Fallback | L1のみモード | 可用性保証 |
