# 1. Phase 4A vs Phase 4B の違い

| 項目 | Phase 4A (Design) | Phase 4B (Implementation) |
|------|-------------------|---------------------------|
| **目的** | デザイン作成・承認 | 実装・統合 |
| **成果物** | Figma + HTML Mock | 実装コード + API統合 |
| **レビュー** | CDO, Marketing, Legal, Personas | 全Agentペルソナ（11名） |
| **PIR頻度** | 週2回可能 | 週1回 |
| **所要時間** | 60-90分 | 90-120分 |
| **ゲート** | デザイン承認 → Phase 4B開始 | 実装承認 → 次週へ |
