# Part 1: ジャーニー概要

## 1.1 全プレイヤージャーニー一覧

| プレイヤー | ジャーニーフェーズ |
|-----------|-------------------|
| End User | 認知 → 理解 → 登録 → 初回Lock → Unlock → 継続利用 → 退会 |
| Token Holder | 認知 → 理解 → veQS Lock → 投票/委任 → 報酬管理 → veQS Unlock |
| Delegate | 認知 → 理解 → 登録 → 委任受け → 投票活動 → 報酬獲得 → 退会 |
| Proposer | 認知 → 理解 → veQS Lock → フォーラム議論 → 提案作成 → 投票 → 実行 |
| Prover | 認知 → 理解 → 申請 → 審査 → Stake → 稼働 → 運用 → 退出 |
| Observer | 認知 → 理解 → Stake → 監視 → Challenge → 報酬 → 退出 |
| Security Council | 任命 → オンボーディング → 監視 → 緊急対応 → 定期レビュー → 退任 |
| Purpose Committee | 任命 → オンボーディング → 理念監視 → Veto検討 → 教育 → 退任 |
| Service Provider | 認知 → 問い合わせ → 契約 → オンボーディング → 運用 → 解約 |
| QS Staff | 入社 → オンボーディング → 権限取得 → 日常運用 → 緊急対応 → 退社 |

```
│  4年ロック時: 1000 × (4/4) = 1000 veQS                                         │
│                                                                                 │

```
